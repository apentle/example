# example
Blog app example

## Setup

1. **Clone the repo**

  ```
  $ git clone https://github.com/apentle/example.git
  $ cd example
  ```

2. **Install dependencies**

  ```
  $ npm install
  ```

3. **Running on Android**

  ```
  $ react-native run-android
  ```

4. **Running on iOS**

  ```
  $ react-native run-ios
  ```
